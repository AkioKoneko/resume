Selected Systems
Talent System
Rebuilt the talent system into a core progression pillar. Per-class and per-guild trees with daily-capped multi-source progression. Talents modify skill behavior and create real build diversity. Build switching is supported but acts as a controlled resource sink.
Competitive Guild Content
Co-designed the competitive guild raid system and owned all encounter design. Seasonal raid showdowns with up to 15 difficulty tiers, leaderboard-driven competition, and staged mechanic escalation. Strong economy sink through consumable and preparation item demand.
Class Ecosystem & Live Balance
Managed live balance across 20 classes in a game with deep player investment. Strengthened alternatives over direct nerfs, used gameplay-loop adjustments alongside stat changes, and tracked community sentiment, class activity, and monetization signals to guide decisions.
Live Content Design
Designed and iterated on seasonal events, dungeons, and repeatable features across a regular live update cadence with ongoing post-release support and tuning.
Catacombs
Designed a session-based PvPvE extraction mode. Players earn loot through PvE objectives but risk losing it on death. Rewards are secured only through successful extraction before a timer expires. Includes lobby registration, group-based hostility, and a dedicated reward storage system.
Non-Combat Guilds
Designed three daily progression systems (Adventurers, Assassins, Mages), each with a unique gameplay loop and gated access through guild talent trees. Guild talents extend buildcrafting with class-specific effects, creating a second progression layer beyond core class talents.
